- [ ] MetaMask connected to Arc Testnet with USDC balance
- [ ] Backend running (localhost:3001)

## 0:00-0:15 — Problem

**Screen**: /drive page, Istanbul map, vehicles moving

**Say**: *"Current navigation updates traffic every few minutes using aggregated phone data. But look at this map — 40 municipal vehicles moving through Istanbul right now. ## 0:15-0:30 — Solution

**Screen**: Type "Taksim" → "Kadikoy" in search bar, red baseline route draws

**Say**: *"CityPulse changes that. Standard route says 31 minutes. But I can query real-time data from municipal vehicles on this corridor — through x402 nanopayments on Arc. Sub-cent USDC, instant finality."*

**Screen**: Point at the two routes on map — red dashed (normal) and cyan glow (CityPulse)
